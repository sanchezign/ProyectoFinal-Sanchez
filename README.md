# ProyectoFinal+Sanchez

## Descripción
Simulador interactivo de Tienda Online en JavaScript. Cumple con los requisitos del proyecto final: consumo de API remota, bibliotecas externas (Bootstrap, SweetAlert2), interactividad asíncrona, etc.

## Instrucciones
1. Abre `index.html` en un navegador.
2. Para desarrollo, usa un servidor local (e.g., Live Server en VS Code) para evitar problemas con fetch.

## Tecnologías
- HTML5
- CSS3 (con Bootstrap)
- JavaScript (ES6+)

- API: Fake Store API
